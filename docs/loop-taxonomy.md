# Loop Taxonomy

Each OfOne loop needs a type, detection cue, linked edges, and failure mode.

| Loop Type | Detection Cue | Common Failure Mode |
|---|---|---|
| reinforcing | change amplifies itself | runaway conclusion, escalating risk |
| balancing | correction dampens change | false stability, hidden backlog |
| measurement | metrics alter behavior | metric gaming, Goodhart pressure |
| incentive | rewards shift moves | local optimization, moral hazard |
| learning | feedback changes future action | overfitting, stale lesson transfer |
| contradiction | conflicting claims force update | unresolved conflict hidden in prose |
| review | gate changes permission or scope | symbolic approval without control |
| deception | adversary manipulates observations | false evidence, poisoned context |
| regime | operating rules change | stale map, wrong adapter projection |

Loop maps make dynamics inspectable rather than buried in narrative.
